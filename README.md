
# slcorona

Daily summary of the confirmed Coronavirus (COVID-19) cases in Sri Lanka

## Installation

You could install the development version from Github using

``` r
# install.packages("devtools")
devtools::install_github("thiyangt/slcorona")
library(slcorona)
```

## Example


``` r
library(slcorona)
## basic example code
data(newpatients)
head(newpatients)
```
